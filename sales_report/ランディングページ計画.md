# ハーゲンダッツ売上レポート ランディングページ計画書 🍦

## 📋 プロジェクト概要

### 目的

2024 年 6 月度ハーゲンダッツ売上データをビジュアル化したランディングページの作成

### 作業フォルダ

`sales_report`

### 技術構成

- **フロントエンド**: HTML（1 ページ構成）
- **グラフライブラリ**: Chart.js
- **データ形式**: CSV（sales_summary.csv, sales_target.csv）
- **スタイリング**: CSS（インライン or 埋め込み）

## 🎯 機能要件

### 必須機能

1. **CSV データ自動読み込み**

   - ページロード時に`sales_summary.csv`と`sales_target.csv`を自動読み込み
   - JavaScript の Fetch API を使用

2. **データ可視化（Chart.js）**

   - 商品別売上実績グラフ（棒グラフ）
   - エリア別売上構成比（円グラフ）
   - 目標達成率比較（横棒グラフ）
   - 前月比較（線グラフ）

3. **サマリー情報表示**
   - 総売上金額：53,550 円
   - 総販売数量：435 個
   - 目標達成率：99.2%

## 📈 グラフ詳細設計

### 1. 商品別売上実績（棒グラフ）

- **データソース**: sales_summary.csv（2024-06 データ）
- **X 軸**: 商品名（バニラ、ストロベリー、チョコレート）
- **Y 軸**: 売上金額
- **色分け**: 商品ごとに異なる色

### 2. エリア別売上構成比（円グラフ）

- **データソース**: sales_summary.csv（区別集計）
- **表示項目**: 渋谷区、港区、中央区、豊島区、新宿区
- **色分け**: 暖色系グラデーション

### 3. 目標達成率比較（横棒グラフ）

- **データソース**: sales_summary.csv + sales_target.csv
- **表示項目**: 商品別達成率（%）
- **基準線**: 100%ライン表示
- **色分け**: 100%以上は緑、未満は赤

### 4. 前月比較（線グラフ）

- **データソース**: sales_summary.csv（2024-05 vs 2024-06）
- **X 軸**: 商品名
- **Y 軸**: 売上金額
- **2 本の線**: 5 月（点線）、6 月（実線）
