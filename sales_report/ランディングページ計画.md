## 📊 表示するグラフ

### 1. 月次売上推移（Line Chart）
- **データ**: 5月→6月の売上金額・販売数量
- **軸**: X軸=月、Y軸=金額/数量
- **色**: 売上=青、数量=緑

### 2. 商品別売上比較（Bar Chart）
- **データ**: 6月の商品別売上金額
- **軸**: X軸=商品名、Y軸=売上金額
- **色**: グラデーション（バニラ=最高値、抹茶=最低値）

### 3. エリア別売上分布（Pie Chart）
- **データ**: 区別売上構成比
- **ラベル**: 区名 + 構成比（%）
- **色**: 渋谷区=強調色、その他=淡色

### 4. 目標達成率（Horizontal Bar Chart）
- **データ**: 商品別目標達成率
- **軸**: X軸=達成率（%）、Y軸=商品名
- **色**: 達成率に応じた段階的色分け

## 🔧 技術実装詳細

### CSVデータ読み込み処理
```javascript
// 画面ロード時にCSVを読み込み
window.addEventListener('DOMContentLoaded', async () => {
    const response = await fetch('sales_summary.csv');
    const csvText = await response.text();
    const data = parseCSV(csvText);
    renderCharts(data);
});
```
